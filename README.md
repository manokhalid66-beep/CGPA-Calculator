# CGPA Calculator - CodeAlpha Internship Task

## Overview

This project is a C++ based CGPA Calculator developed as part of the CodeAlpha C++ Programming Internship. The application allows students to calculate their CGPA by entering course grades and credit hours. The system automatically converts grades into grade points, calculates total grade points, computes the CGPA, and displays the student's academic performance.

## Features

* User-friendly console interface
* Input multiple courses
* Grade to grade-point conversion
* Automatic CGPA calculation
* Total credit hours calculation
* Performance evaluation based on CGPA
* Accurate results with decimal precision
* Error-free and efficient implementation

## Technologies Used

* C++
* Object-Oriented Programming Concepts
* Functions
* Loops
* Conditional Statements
* String Handling
* Input/Output Formatting

## How It Works

1. Enter the number of courses.
2. Enter the grade and credit hours for each course.
3. The program converts grades into grade points.
4. Total grade points and credits are calculated.
5. CGPA is computed automatically.
6. The final CGPA and performance status are displayed.
